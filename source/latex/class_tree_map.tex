\subsection{Detailed Description}
\subsubsection*{template$<$class K, class V$>$class Tree\-Map$<$ K, V $>$}

\hyperlink{class_tree_map}{Tree\-Map} is the balanced-\/tree implementation of map. The iterators must iterate through the map in the natural order (operator$<$) of the key.
